Biggest "ohh!" Moments

1.

Array methods exist because each one solves a specific real-world problem.

Instead of memorizing methods, I now ask:

> "What problem was this method invented to solve?"

2.

Projects teach methods better than isolated exercises.

3.

Some methods change memory.
