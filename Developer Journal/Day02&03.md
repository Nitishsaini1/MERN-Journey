---

Bug 2

Used = instead of ==.

Learned

= stores.

---

Bug 3

Thought

score + 1

would change memory.

Learned

Calculating is different from storing.


---

Bug 4

Compared empty input with null.

Learned

Empty input is stored as an empty string ("").
