📖 Day 05 Journal

Date: 08 July 2026

What I learned

Today was one of the biggest learning days I've had.

I understood that a function doesn't automatically give its result back. return exists to send a value back to whoever called the function. I finally understood the difference between console.log() and return.

I learned how JavaScript reads code, remembers function definitions, and only executes them when they are called.

I also learned about variables using let and const, reassignment, scope, and how JavaScript searches for variables depending on where it is currently executing.

I understood how && and || work by thinking about conditions instead of memorizing them.

The biggest practical achievement was building my first JavaScript project: a Guess the Number game. During the project I learned to:

Update the webpage using textContent.

Count attempts.

Disable the submit button after winning.

Reset the game.

Generate a random number using Math.random().


I also learned that debugging is about asking:

What do I know is working?

What should happen next?

Where does the data live?

Can I prove my assumption?


That way of thinking helped me solve bugs instead of randomly changing code.

Mistakes I made

Confused "Too High" and "Too Low".

Reset the webpage but forgot to reset the JavaScript variable.

Wrote * 1 instead of + 1 while learning Math.random().

Forgot that changing a JavaScript variable does not automatically update the webpage.


Biggest realization

I don't learn well by memorizing definitions.

I learn much better by asking why, reasoning through the problem, and becoming JavaScript in my mind.

Today I felt like I started thinking like JavaScript instead of just writing JavaScript.
